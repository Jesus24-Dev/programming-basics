# Object Relational Maping (ORM)
Es una tecnica de programacion que permite convertir datos entre sistemas incompatibles utilizando lenguajes de programacion orientada a objetos. En las bases de datos, un ORM facilita la interaccion con una base de datos relacional mediante la abstraccion de las operaciones SQL en forma de metodos de ojetos.

## Caracteristicas de un ORM

### Abstraccion SQL
Los desarrolladores pueden interactuar con la base de datos utilizando el lenguaje de programacion en vez de escribir consultas SQL manuales

### Mapeo de tablas a clases
Las tablas de la base de datos se mapean a clases en el lenguaje de programacion. Cada fila en una tabla se convierte en una instancia de la clase correspondiente

### Operaciones CRUD simplificadas
Crear, leer, actualziar y borrar se realizan mediante metodos de objetos.

### Gestion de relaciones
Los ORM manejan relaciones entre tablas (como uno a muchos, muchos a muchos) de una manera que se integra naturalmnete con programacion orientada a objetos.
